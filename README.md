# Galaxy-Website
This is a simple Galaxy themed website created using **HTML, CSS, and Video Background**.  
It contains a navigation bar, background video, title text and image gallery section.

## Features
- Background space video
- Navigation bar
- Explore button
- Image gallery using Flexbox
- Responsive look
- Clean UI

## Technologies Used
- HTML
- CSS
- Flexbox
- Video Background

## Demo
Screen recording of the project is also uploaded in this repository.

## Conclusion
This Galaxy Website project shows how HTML and CSS can be used to design a beautiful space-themed web page. It includes a background video, navigation bar, heading section and image gallery, which helps in understanding layout design, flexbox, styling and webpage structure. This project improves web design skills and gives practical experience in frontend development.
